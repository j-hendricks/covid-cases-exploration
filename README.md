## Overview
Explore COVID-19 cases in the United States using the [Delphi EpiData API](https://cmu-delphi.github.io/delphi-epidata/).

## Scope

### Endpoints
The following endpoints are used:

- CovidCast: https://api.covidcast.cmu.edu/epidata/api.php
- Covid Hospitalization Time Series: https://api.delphi.cmu.edu/epidata/covid_hosp_state_timeseries/ 

### Locations
Current locations included are as follows:

- Virginia
- District of Columbia
- Maryland
- Pennsylvania

### Timeframe
Data occurred between 2020-01-01 and 2025-12-31.
